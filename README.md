# Beyond-the-Pyramids-tourism-website-
This project is a role-based tourism hub for Egypt, structured around two primary interfaces: a User Dashboard and an admin Dashboard. The platform begins with a Landing Page that introduces the website, alongside Registration and Login
pages for account creation and authentication. An Error Page handles system-level issues including 403, 404, and 500 responses. The User Dashboard serves as the main hub after login, offering tourists four trip categories: Single Location
trips, Full-Day packages, Full-Week packages, and a Custom Trip Builder where users can configure their own itinerary by selecting locations & accommodation. A dynamic Package Details Page presents relevant information for any selected
package. The booking flow proceeds through a Travelers Details page, a Booking Summary with payment confirmation, and a Booking Details page displaying the final booking number. Beyond booking, users can manage their experience
through a User Profile page, a My Bookings page, and a Customer Review page for submitting feedback. Supporting pages include FAQ, Contact Us, About Us, and Terms & Conditions. The Admin Dashboard provides full platform
oversight through four dedicated modules: Booking Management for monitoring and cancelling bookings, Package Management for maintaining trip offerings, User Management for overseeing all registered accounts, and Reports &
Analytics for tracking website performance and generating insights.
